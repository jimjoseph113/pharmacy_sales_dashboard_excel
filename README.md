 OVICO Pharmacy Dashboard

Project Overview

This project is an interactive Excel dashboard created to analyze pharmacy sales performance, inventory status, product profitability, and monthly profit trends.

The dashboard provides business insights that help pharmacy managers make better inventory and revenue decisions.


Objectives

- Track total revenue
- Analyze profit by product
- Monitor monthly profit trends
- Identify expiring products
- Track remaining stock levels


 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Data Cleaning Techniques

---

 Dashboard Features

 KPIs
- Total Revenue
- Total Profit Earned
- Total Units Sold
- Total Inventory Cost

 Visualizations
- Revenue by Category
- Profit by Product
- Monthly Profit Trend
- Stock Remaining
- Expiring Products


 Process

1. Data Cleaning
2. Data Formatting
3. Pivot Table Creation
4. KPI Calculations
5. Dashboard Design
6. Interactive Slicer Integration


Insights

- Some products generated high revenue but low profit margins.
- Inventory levels for certain products are running low.
- Monthly profit fluctuated significantly across the year.
- A few products are approaching expiration and require quick sales action.
